# Инструкция для друзей: как открыть Pulse Desk

Эта инструкция для тех, кто хочет просто посмотреть Pulse Desk через браузер. Ничего устанавливать не нужно.

## Что вам должен прислать владелец

Владелец должен прислать две вещи:

1. HTTPS-ссылку вида:

```text
https://something.trycloudflare.com
```

2. Отдельный пароль для просмотра:

```text
VIEWER_TOKEN
```

`ADMIN_TOKEN`, файлы `.env`, `.session`, `.db`, `app.log` и папку `backups/` владелец не отправляет.

## Как зайти

1. Откройте присланную HTTPS-ссылку в браузере.
2. На экране входа вставьте `VIEWER_TOKEN`.
3. Нажмите вход.
4. Если страница открылась, можно смотреть дашборд, упоминания, розыгрыши, аналитику и рынок.

Друзья входят только с `VIEWER_TOKEN`. Этот режим предназначен для просмотра, без управления аккаунтами и настройками владельца.

## Что не нужно делать

- Не устанавливайте Python.
- Не запускайте `main.py`.
- Не скачивайте Telegram session-файлы владельца.
- Не просите у владельца `ADMIN_TOKEN`.
- Не вводите ссылку с токеном в адресной строке. Токен вводится только на странице входа.
- Не отправляйте никому коды входа Telegram.

## Если ссылка не открывается

Проверьте по порядку:

1. Ссылка должна начинаться с `https://`.
2. Откройте ссылку в обычном браузере, не внутри превью мессенджера.
3. Обновите страницу через `Ctrl+F5`.
4. Проверьте, что `VIEWER_TOKEN` вставлен без пробелов в начале и конце.
5. Если браузер пишет, что сайт недоступен, значит у владельца остановлен локальный сервер или Cloudflare Tunnel. Напишите владельцу, чтобы он заново запустил шаринг.

## Что должен сделать владелец, чтобы ссылка работала

На компьютере владельца в папке проекта:

```powershell
.\share_with_friends.ps1
```

Скрипт запускает локальный Pulse Desk и Cloudflare Quick Tunnel. В терминале появится HTTPS-ссылка `*.trycloudflare.com`; её и нужно отправить друзьям.

Важно:

- окно PowerShell со скриптом должно оставаться открытым;
- если закрыть окно, ссылка перестанет работать;
- друзьям отправляется только HTTPS-ссылка и `VIEWER_TOKEN`;
- `ADMIN_TOKEN` остаётся только у владельца.

## Короткое сообщение, которое можно отправить другу

```text
Открой ссылку:
https://....trycloudflare.com

На странице входа вставь VIEWER_TOKEN, который я пришлю отдельно.
Ничего устанавливать не нужно. Если не открывается, попробуй Ctrl+F5 или напиши мне - возможно, я перезапущу ссылку.
```
